# MTA Metro Data Hub (Frontend)

Frontend UI for generating custom Excel reports from the MTA Flood API.

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - build for production
- `npm run preview` - preview production build

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
